# Perspective Crop

Perspective Crop is a single-page HTML tool for selecting four points on an image, correcting perspective, previewing the straightened result, adjusting brightness/contrast, and downloading the output as JPG.

The app runs entirely in the browser. It uses OpenCV.js from the official CDN, so an internet connection is required unless you host `opencv.js` locally.


```text
https://ngvthangbz.github.io/CropImage/
```
